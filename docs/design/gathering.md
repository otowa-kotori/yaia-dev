## 1. 核心概念

角色有三种互斥的工作状态：**战斗**、**采集**、**生产**。同一时刻只能做一件事。玩家的核心决策是分配角色到不同岗位。

### 四种技能

| 技能 | 类型 | 说明 |
|------|------|------|
| 采矿 | 采集 | 在矿脉上挂机，产出矿石 |
| 伐木 | 采集 | 在树上挂机，产出木材 |
| 锻造 | 生产 | 在锻造台挂机，消耗材料产出金属制品（武器、护甲、半成品等）|
| 木工 | 生产 | 在木工台挂机，消耗材料产出木制品（弓、木柄、木板等）|

采集和生产**都是持续挂机活动**，都占用角色时间槽，都有技能等级和经验。

### 采集不绑定角色

任何角色都可以学任何技能、做任何工作。采集/生产效率只取决于**技能等级**和**装备**，与角色职业/属性无关。

## 2. 采集

### 机制

角色分配到一个采集点（矿脉/树木），开始持续挂机采集。每 `swingTicks` 完成一次挥动，掷骰掉落表产出原材料，同时获得采集技能经验。

```
角色 → 选择采集点 → 每 swingTicks 产出原材料 + 技能经验
                    ↑ 持续循环直到玩家中断
```

### 采集点

前期两种采集线，各有低级/高级采集点：

| 采集点 | 技能 | 等级要求 | 产出 | 说明 |
|--------|------|----------|------|------|
| 铜矿脉 | 采矿 | 1 | 铜矿石 | 第一个矿点 |
| 铁矿脉 | 采矿 | 待定 | 铁矿石 | 高级矿点 |
| 普通树 | 伐木 | 1 | 木材 | 第一个伐木点 |

> 采集点的具体 swingTicks、经验量、是否需要更多种类的木材，待数值联调后确定。

### 效率因素

- **硬性门槛**：技能等级——未达到等级的采集点无法采集
- **效率因素**：采集装备——更高 Tier 的镐/斧 → swingTicks 缩短

采集装备本身需要生产制作，形成"做工具→高效采集→做更好装备"的自举循环。

### 采集经验

高级采集点给更多技能经验，激励玩家推进到更高级的资源。

## 3. 生产

### 机制

生产和采集一样是**角色挂机活动**。角色分配到一个生产台（锻造台/木工台），选择一个配方，开始持续生产。每完成一次生产消耗输入材料、产出成品/半成品、获得生产技能经验。

```
角色 → 选择生产台 → 选择配方 → 每 durationTicks 消耗材料 + 产出物品 + 技能经验
                               ↑ 持续循环直到材料耗尽或玩家中断
```

**关键设计**：
- 生产**有时间成本**，和采集一样占用角色时间
- 材料不足时自动停止
- 技能等级不够时无法选择该配方

### 生产台

| 生产台 | 技能 | 说明 |
|--------|------|------|
| 锻造台 | 锻造 | 生产金属类物品：铜锭、铁锭、金属武器、金属护甲 |
| 木工台 | 木工 | 生产木制品：木板、弓、木柄、部分采集工具 |

### 配方结构

每个配方有：
- 所属技能 + 最低等级要求
- 输入材料列表（种类 + 数量）
- 输出物品列表（种类 + 数量）
- 生产时间（durationTicks）
- 经验奖励

## 4. 生产链

### T0→T1：原材料直接合成

```
铜矿石 ──→ 铜武器 / 铜防具 / 铜镐 / 铜斧
木材   ──→ 弓 / 木制装备
```

T1 不需要半成品，原材料直接合成。链条短，教学友好。

### T2：需要半成品

```
铜矿石 + 铁矿石 ──锻造──→ 铁锭（半成品）
木材            ──木工──→ 木板（半成品）

铁锭 + 木板 + 怪物材料 ──锻造/木工──→ 铁武器 / 铁防具 / 铁采集装备
```

T2 引入半成品，玩家需要先让一个角色挂机生产铁锭/木板，再用这些半成品合成最终装备。

**时间成本体验**：
1. 角色 A 采矿 → 产出铁矿石（采集时间）
2. 角色 B 伐木 → 产出木材（采集时间）
3. 角色 A 或 B 锻造铁锭 ← 消耗铁矿石（生产时间）
4. 角色 A 或 B 木工木板 ← 消耗木材（生产时间）
5. 某角色锻造铁剑 ← 消耗铁锭 + 木板 + 怪物材料（生产时间）

每一步都是挂机时间。玩家需要规划角色分工和生产顺序。

### 半成品

| 半成品 | 生产台 | 原料 | 用途 |
|--------|--------|------|------|
| 铁锭 | 锻造台 | 铁矿石 × N | 铁制武器/护甲/采集装备合成 + 强化 |
| 木板 | 木工台 | 木材 × N | 弓/部分装备合成 + 强化 |

新手阶段只有 2 种半成品，链条最多两步。

## 5. 材料总览

### 原材料

| 材料 | 来源 | 用途 |
|------|------|------|
| 铜矿石 | 采矿（铜矿脉）| T1 装备合成，T2 装备强化辅材 |
| 铁矿石 | 采矿（铁矿脉）| 锻造铁锭 |
| 木材 | 伐木 | T1 装备合成，木工木板 |

### 半成品

| 材料 | 来源 | 用途 |
|------|------|------|
| 铁锭 | 锻造（铁矿石）| T2 装备合成 |
| 木板 | 木工（木材）| T2 装备合成 |

### 怪物掉落

| 材料 | 来源 | 用途 |
|------|------|------|
| 史莱姆胶 | 史莱姆掉落 | T1 装备辅材，T2 强化辅材 |
| （其他怪物材料 ×2-3）| 各区域怪物 | 不同装备的合成/强化配方 |
| Boss 材料 | Boss 副本 | 高级装备/特殊合成 |

> 具体怪物材料种类和名称待 monsters 模块确定后回填。

### 总量

约 10 种物品（3 原材料 + 2 半成品 + 4-5 怪物/Boss 材料）。"每种材料至少 2 个用途"的原则确保低级材料永远有用。

## 6. 角色分工示意

以 4 角色为例，不同阶段的典型分工：

**阶段 1（2 角色）**：
```
骑士：打怪（练级 + 怪物材料）
骑士：采矿 → 生产铜装备（交替进行）
```

**阶段 2-3（3 角色）**：
```
角色 A：打怪
角色 B：采矿
角色 C：伐木
→ 积累足够材料后，让 B 或 C 去锻造/木工
```

**阶段 4（4 角色）**：
```
角色 A+B：打怪（组队刷高级怪）
角色 C：采矿（铁矿脉）
角色 D：伐木 → 锻造铁锭 → 交替
```

玩家始终在权衡：**谁去打怪谁去采集谁去生产**。角色是稀缺资源。

## 7. 与心流的对应

| 心流阶段 | 采集/生产内容 | 教学 |
|---------|-------------|------|
| 阶段 0（Lv1-3） | 无 | 纯战斗 |
| 阶段 1（Lv3-8） | 采矿 + 直接合成 T0/T1 装备 | 采集→生产→装备的基础循环 |
| 阶段 2（Lv8-14） | 伐木解锁，开始采铁矿 | 多角色分工，第二种采集 |
| 阶段 3（Lv14-20） | 生产半成品（铁锭/木板），合成 T2 | 完整生产链 |
| 阶段 4-5（Lv20-30） | T2 装备强化为主 | 深度材料消耗 |
