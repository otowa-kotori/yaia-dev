## 1. 核心概念

角色有三种互斥的工作状态：**战斗**、**采集**、**生产**。同一时刻只能做一件事。玩家的核心决策是分配角色到不同岗位。

### 四种技能

| 技能 | 类型 | 说明 |
|------|------|------|
| 采矿 | 采集 | 在矿脉上挂机，产出矿石 |
| 伐木 | 采集 | 在树上挂机，产出木材 |
| 锻造 | 生产 | 在锻造台挂机，消耗材料产出金属制品 |
| 木工 | 生产 | 在木工台挂机，消耗材料产出木制品 |

采集和生产**都是持续挂机活动**，都占用角色时间槽，都有技能等级和经验。

### 采集不绑定角色

任何角色都可以学任何技能、做任何工作。效率只取决于**技能等级**和**装备**，与角色职业/属性无关。

## 2. 采集

### 机制

角色分配到一个采集点，开始持续挂机采集。每 `swingTicks` 完成一次挥动，根据**采集力**判定结果，同时获得采集技能经验。

```
角色 → 选择采集点 → 每 swingTicks 判定一次采集力
                    ↑ 持续循环直到玩家中断
```

### 采集力（Gathering Power）

采集力 = f(技能等级, 采集装备)，和采集点难度做比较。

**公式**：`期望产出 = cap × r^p / (r^p + cap - 1)`，其中 `r = 采集力 / 难度`。

采集力参数：**cap=4, p=2**。

> 脚本：`docs/design/scripts/gathering_power_curve_v2.py`  
> 图表：`docs/design/plots/gathering_power_curve_v2.png`

| r（力/难度比） | 期望产出 | 体感 |
|---------------|---------|------|
| 0.5 | 0.31 | 大部分 swing 失败 |
| 0.7 | 0.56 | 约半数失败 |
| 1.0 | **1.00** | 达标，稳定 1 个 |
| 1.5 | 1.71 | 经常出 2 个 |
| 2.0 | 2.29 | 期望 2+ |
| 3.0 | 3.00 | 期望 3 |
| ∞ | → 4.00 | 理论上限 |

**4 倍是长线养成目标**，新手世界内 r 最高约 2-3，对应 ~2 倍产出。已经有明显收益感，但远没到天花板。后续世界的高级装备和技能等级继续推高 r。

**公式性质**：
- r=1 时恰好 = 1.0（数学保证）
- r<1 时自然衰减——不需要单独的"失败"逻辑，期望 <1 就是"平均不到 1 个"
- 一个公式统一处理失败和多产，没有分段

**实现**：每次 swing 掷骰决定产出 0/1/2/3/4 个，使期望值等于公式值。

**设计意图**：
- 低等级去高级采集点 → 频繁失败 → 自然门槛，不需要硬性禁止
- 高等级回低级采集点 → 多产 → 快速清库存
- 采集装备提供采集力加成 → 装备有实际价值
- 长线成长 → 永远有提升空间

### 采集点

**采矿线**：

| 采集点 | 技能 | 难度 | 产出 | 说明 |
|--------|------|------|------|------|
| 铜矿脉 | 采矿 | 低 | 铜矿石 | 第一个矿点 |
| 铁矿脉 | 采矿 | 中 | 铁矿石 | T2 装备主材料 |
| 银矿脉 | 采矿 | 高 | 银矿石 | 仅用于 T2 装备强化 |

**伐木线**：

| 采集点 | 技能 | 难度 | 产出 | 说明 |
|--------|------|------|------|------|
| 橡木 | 伐木 | 低 | 橡木原木 | 第一个伐木点 |
| 桦木 | 伐木 | 中 | 桦木原木 | T2 装备主材料 |
| 松木 | 伐木 | 高 | 松脂 | 仅用于 T2 装备强化 |

**第三级采集点的产出只用于装备强化**，不参与装备合成。松木产出的是松脂而非原木——它是一种强化辅料，不是建材。

### 采集经验

高级采集点给更多技能经验。采集失败只给极少量经验（约正常的 5%），**不鼓励靠失败练级**——玩家应该提升采集力（升级技能 + 换更好的采集装备）后再挑战高级采集点。

## 3. 生产

### 机制

生产和采集一样是**角色挂机活动**。角色分配到生产台，选择配方，开始持续生产。每完成一次生产消耗输入材料、产出成品/半成品、获得生产技能经验。

```
角色 → 选择生产台 → 选择配方 → 每 durationTicks 消耗材料 + 产出物品 + 技能经验
                               ↑ 持续循环直到材料耗尽或玩家中断
```

**关键设计**：
- 生产**有时间成本**，和采集一样占用角色时间
- 材料不足时自动停止
- 技能等级不够时无法选择该配方

### 生产力（Crafting Power）

生产力 = f(技能等级, 生产装备/工具)。**生产力越高，durationTicks 越短**。

**公式**：`速度倍率 = cap × r^p / (r^p + cap - 1)`，其中 `r = 生产力 / 配方难度`。

生产力参数：**cap=8, p=3**。实际 durationTicks = baseDurationTicks / 速度倍率。

| r（力/难度比） | 速度倍率 | 体感 |
|---------------|---------|------|
| 1.0 | 1.0x | 达标，基础速度 |
| 1.2 | 1.6x | 明显加速 |
| 1.5 | 2.6x | 进度条快了一倍多 |
| 2.0 | 4.3x | 新手阶段极限，接近 4 倍速 |
| 3.0 | 6.4x | 后续世界目标 |
| ∞ | → 8.0x | 理论上限 |

**和采集力的设计差异**：

采集力和生产力共用公式结构 `cap × r^p / (r^p + cap-1)`，但参数不同：

| | 采集力 | 生产力 |
|---|--------|--------|
| cap | 4 | 8 |
| p | 2 | 3 |
| 新手 r≈2 | ~2.3 倍产出 | ~4.3 倍速 |
| 有失败？ | 有（r<1 产出<1）| 无 |

- 采集有失败惩罚 + 低 cap + 慢增长 → 多产是**长线锦上添花**
- 生产无失败 + 高 cap + 快增长 → 速度提升是**核心成长回报**

生产**不会失败**——只要技能等级够选配方，就一定能做出来，只是速度不同。

### 生产台

| 生产台 | 技能 | 说明 |
|--------|------|------|
| 锻造台 | 锻造 | 金属类：矿锭、金属武器、金属护甲、金属采集工具 |
| 木工台 | 木工 | 木制品：木板、弓、木制装备、木制采集工具 |

### 配方结构

每个配方有：
- 所属技能 + 最低等级要求
- 输入材料列表（种类 + 数量）
- 输出物品列表（种类 + 数量）
- 基础生产时间（baseDurationTicks，受生产力缩短）
- 经验奖励

## 4. 生产链

### T0→T1：原材料直接合成

```
铜矿石 + 怪物材料 ──锻造──→ 铜武器 / 铜防具 / 铜镐
橡木原木          ──木工──→ 铜斧（木柄部分）/ 弓
```

T1 不需要半成品，原材料直接合成。链条短，教学友好。

### T2：需要半成品

```
铁矿石          ──锻造──→ 铁锭（半成品）
桦木原木         ──木工──→ 桦木板（半成品）

铁锭 + 桦木板 + 怪物材料 ──锻造/木工──→ 铁武器 / 铁防具 / 铁采集装备
```

T2 引入半成品，玩家需要先让角色挂机生产铁锭/桦木板，再用这些半成品合成最终装备。

### 练级半成品

**问题**：装备做完后，生产技能没有"值得做"的配方了，但技能等级还需要继续升。

**解决**：设计一批**低消耗、有用途、主要价值是给经验**的练级配方。

| 练级半成品 | 生产台 | 输入 | 用途 | 设计意图 |
|-----------|--------|------|------|---------|
| 铜钉 | 锻造 | 铜矿石（少量）| T1/T2 强化辅材，部分合成辅材 | 低级锻造练级物 |
| 铁环 | 锻造 | 铁矿石（少量）| T2 强化辅材 | 中级锻造练级物 |
| 木楔 | 木工 | 橡木原木（少量）| T1/T2 强化辅材，部分合成辅材 | 低级木工练级物 |
| 桦木柄 | 木工 | 桦木原木（少量）| T2 武器/工具合成辅材 | 中级木工练级物 |

**特点**：
- 消耗**常见便宜材料**，不和装备配方抢稀缺怪物素材
- 产出的东西本身**有实际用途**（强化辅材、合成辅材），不是纯废品
- 经验/时间比不如做装备高，但**材料容易获取**，适合持续挂机
- 一种材料永远有用：铜矿石做铜钉→铜钉用于 T2 强化

### 总量

约 **20 种**物品：
- 6 采集原材料（铜/铁/银矿石 + 橡木/桦木原木 + 松脂）
- 6 半成品（2 装备半成品 + 4 练级半成品）
- 8 怪物素材

"每种材料至少 2 个用途"的原则确保低级材料永远有用。秘银矿石和杉木原木是例外——它们只用于高级强化，但这是有意的：作为高级采集的**持续目标**存在。

## 6. 角色分工示意

以 4 角色为例，不同阶段的典型分工：

**阶段 1（2 角色）**：
```
角色 A：打怪（练级 + 怪物材料）
角色 B：采矿 → 锻造铜装备（交替进行）
```

**阶段 2-3（3 角色）**：
```
角色 A：打怪
角色 B：采矿（或锻造铜钉练级）
角色 C：伐木（或木工木楔练级）
→ 积累足够材料后切去做装备
```

**阶段 4（4 角色）**：
```
角色 A+B：打怪（组队刷高级怪，攒怪物素材）
角色 C：采矿铁矿 → 锻造铁锭/铁环 → 交替
角色 D：伐木桦木 → 木工桦木板/桦木柄 → 交替
```

玩家始终在权衡：**谁去打怪谁去采集谁去生产**。角色是稀缺资源。

## 7. 与心流的对应

| 心流阶段 | 采集/生产内容 | 教学 |
|---------|-------------|------|
| 阶段 0（Lv1-3） | 无 | 纯战斗 |
| 阶段 1（Lv3-8） | 采矿铜矿 + 伐木橡木 + 直接合成 T0/T1 | 采集力、采集→生产→装备循环 |
| 阶段 2（Lv8-14） | 铁矿/桦木解锁，练级半成品 | 多角色分工，生产力，技能练级 |
| 阶段 3（Lv14-20） | 生产铁锭/桦木板，合成 T2 装备 | 完整生产链 |
| 阶段 4-5（Lv20-30） | 秘银/杉木解锁，T2 装备强化 | 高级采集，深度强化材料消耗 |
