# 采集与生产

> 依赖：[equipment.md](./equipment.md), [monsters.md](./monsters.md)  
> 被依赖：无（末端模块）  
> **状态**：方向探索中

## 设计理念

### 前期生产链不宜过深

第一阶段（新手 Lv 1-30）不出现二阶半成品。

```
原材料 → 半成品 → 成品（装备/强化材料）     ← 最多两步
原材料 → 成品（低级装备直接合成）            ← 有些更简单
```

不做：原材料 → 半成品 A → 半成品 B → 成品。

### 采集技能之间完全平行

采矿、伐木、采药是三条独立线。

- 不需要采矿 Lv X 才能解锁伐木
- 不同角色可以专精不同的采集技能
- 锻造（合成技能）可能需要多种材料的产物——这是跨采集技能的唯一交叉点

### 一个物品多种用途

每种材料至少用于 2 个不同目的：

例如铜矿石：
1. 合成铜剑
2. 铁装备的强化材料（作为辅助成分）

这样"低级材料永远有用"自然成立。

### 属性影响采集效率

- 硬性门槛：采集技能等级（必须达到才能采集）
- 软性效率：一级属性影响采集速度

| 采集技能 | 关联属性 | 方向 |
|----------|----------|------|
| 采矿 | STR | 战士相性最高 |
| 伐木 | DEX | 巡林客相性最高 |
| 采药 | INT | 法师/牧师相性最高 |

高属性 → swing 间隔缩短 → 单位时间产出更多。但属性不足也能采，只是效率低。

### 材料总量控制

全新手阶段约 9 种材料（和 monsters 模块同步确定）：
- 采集类：~4 种（2 种矿、1 种木、1 种草药）
- 怪物掉落：~4 种
- Boss 专属：~1 种

加上 1-2 种半成品（铁钉、木板之类），总计约 10-11 种物品。

## 采集节奏方向

挂机游戏的采集节奏应该是"放着就有产出"，但效率和玩家选择有关。

- 低级采集点：swing 快（6-8 秒），产出低级材料
- 高级采集点：swing 慢（10-15 秒），产出高级材料，门槛高
- 属性加成可以把 swing 时间压缩约 30-40%

### 采集经验

高级矿/木给的采集技能经验更多，值得推进到更高级的采集点。

## 半成品方向

| 半成品 | 原料 | 用途 |
|--------|------|------|
| 某种金属件 | 矿石 × N | 武器/护甲合成 + 强化 |
| 某种木件 | 木材 × N | 弓/某些装备合成 + 强化 |

只有 1-2 种半成品，保持链条短。
