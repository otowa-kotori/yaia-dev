## 2. 技能系统规则

- **每个职业 6 个技能**，每个技能 10 级，每级消耗 TP
- **前置关系**：部分技能需要前置技能达到指定等级才能学习
- **TP 来源**：每级 +3 点（部分升级直接获取，部分外部获取），额外来源待定
- **TP 总量设计**：Lv30 时 TP 不足以点满全部 6 技能。这是**有意的稀缺**——每个职业的 6 技能内部分成 A 线 / B 线两个方向，玩家在 Lv30 必须做加点取舍：偏向某一条线、还是均匀混搭。不同选择对应不同的挂机定位（高甲 / 群怪 / 单刷 / 组队）。
- **技能重置**：低成本或免费
- **默认攻击**：每个职业都有一个不消耗资源的默认攻击
  - 骑士：挥武器，物理伤害，系数正常
  - 游侠：射箭，物理伤害，系数正常；沉静射击触发时会周期性替代默认攻击
  - 法师 / 圣女：低系数魔法射线，走 MATK。**伤害刻意压得很低**，仅为"蓝空时不完全站桩"的保底手段；设计上就是让法师 / 圣女在蓝空时 DPS 显著下降，给玩家管 MP 的压力，同时保持"一直在施法"的画风（不会举法杖敲人）

### 技能类型

- **主动**：占战斗回合，由 AI 在 intent 阶段选择释放，消耗 MP 或 CD 限制
- **被动**：常驻效果，不占回合，不需要 AI 判断
- **长效姿态**（特殊主动）：形式上是主动技能，但**持续整场战斗**，开启后不需要再操作。挂机 AI 在战斗开始前根据玩家配置自动开启对应姿态。同类姿态互斥（开了 A 就不能开 B）。本质上是"加点决定的被动选项"，但保留主动开启的形式给手动玩家做切换空间。
  - 用于：骑士狂怒/守护、法师护甲

### MP 管理（挂机模型）

```
每回合 MP 变化 = 基础回复 + 被动回复 - 技能消耗

均衡态：回复 ≥ 消耗 → 永远有技能可放（稳态挂机）
失衡态：回复 < 消耗 → MP 逐渐耗尽 → 退回默认攻击 → 效率降低
```

AI 规则：优先使用技能，MP 不够时自动用默认攻击等待回复。玩家离线前配好装备/技能/食物，确保均衡态。

- 骑士 / 游侠：MP 消耗低，基本不需要管理
- 法师：MP 消耗高，需要投资 MP 回复被动
- 圣女：MP 消耗中等，组队治疗时压力增大

### A 线 / B 线分支说明

每个职业的 6 技能内部隐含两条发展线，体现不同玩法定位。两条线**不强制互斥**——技能本身都能学，但 TP 不够点满，玩家加点时事实上要选边。少数机制（如骑士的两个姿态）是**严格互斥**的。

| 职业 | A 线 | B 线 |
|------|------|------|
| 骑士 | 进攻型（高输出但脆） | 防御型（极限肉，组队保护） |
| 游侠 | 多段流（数量碾压软目标） | 弱点流（精确打击硬目标） |
| 法师 | 火系（纯输出） | 冰系（控场 + 输出） |
| 圣女 | 自立型（单刷强化 Smite） | 团辅型（治疗 + 增益） |

## 3. 骑士 Knight

| 技能 | 英文 | 类型 | 前置 | 分支 | 方向 | 升级效果 |
|------|------|------|------|------|------|----------|
| **重击** | Power Strike | 主动 | 无 | A | 高系数单体物理，无 CD | 系数提升 |
| **战吼** | Warcry | 主动 | 重击 Lv3 | B | 嘲讽全体 + 自身减伤 buff，有 CD | 持续时间 / 减伤% |
| **坚守** | Fortitude | 被动 | 无 | B | +HP% 和 +PDEF% | % 提升 |
| **反击** | Retaliation | 被动 | 重击 Lv3 | A | 被攻击时概率自动反击一次普攻 | 触发概率提升 |
| **狂怒** | Rage | 长效姿态 | 反击 Lv3 | A | 整场战斗 +PATK% / -PDEF%（净增伤但更脆） | +PATK% 提升，-PDEF% 也略提升 |
| **守护** | Guard | 长效姿态 | 坚守 Lv3 | B | 整场战斗 +PDEF% / -PATK%；队友受击时概率代受 | +PDEF% 与代受概率提升 |

**主动 2 : 被动 2 : 姿态 2**（姿态互斥，同时只能开一个）

**狂怒 vs 守护的设计意图**（DND 3.5 的 Power Attack vs Combat Expertise）：
- 狂怒：牺牲防御换攻击，配合反击形成"高输出但脆"的进攻流
- 守护：牺牲攻击换防御 + 帮队友挡刀，配合坚守形成"极限肉 + 组队核心"
- **互斥**：玩家加点偏向哪条线，就开对应姿态。两条都点了也只能开一个 → 重复投资浪费 TP，自然形成抉择
- 数值守门：狂怒的 +PATK% > -PDEF%（净收益正向），但 -PDEF% 在打高输出敌人时是真正的代价；守护反向同理

**战斗公式交互**：
- 狂怒下重击伤害进一步放大，但承伤激增
- 守护下重击伤害打折，但生存性极强

## 4. 游侠 Ranger

| 技能 | 英文 | 类型 | 前置 | 分支 | 方向 | 升级效果 |
|------|------|------|------|------|------|----------|
| **连射** | Rapid Shot | 主动 | 无 | A | 多段物理（2×~65%），短 CD | 每段系数提升 |
| **标记** | Hunter's Mark | 主动 | 连射 Lv3 | B | 降低目标 PDEF，持续数回合，有 CD | 降低量 / 持续时间 |
| **锐利** | Keen Edge | 被动 | 无 | B | +暴击率 +暴击伤害 | % 提升 |
| **毒箭** | Venom Arrow | 主动 | 连射 Lv3 | A | 中系数单体 + DoT（持续伤害），有 CD | 系数 / DoT 伤害提升 |
| **齐射** | Volley | 主动 | 连射 Lv5 | A | AOE 多段（每个敌人各打 1 段 ~50%），长 CD | 系数提升 / CD 降低 |
| **沉静射击** | Steady Shot | 被动 | 锐利 Lv3 | B | 每 N 次普攻后，下一发普攻变为高系数单段 | 系数提升 / N 降低 |

**主动 4 : 被动 2**

**A 线（多段流）vs B 线（弱点流）的取舍**：
- A 线靠**数量**：连射 + 齐射快速清软目标，毒箭 DoT 用固定伤害补救高甲。打小怪海效率最高，但单发系数都低 → 碰到高甲精英时被破甲规则双重惩罚。
- B 线靠**质量**：标记减 PDEF + 锐利堆暴击 + 沉静射击周期高伤单发。每发都打满，专克硬目标，但清群速度慢。
- 加点不够时玩家选边：刷小怪走 A，打 boss / 高甲怪走 B。

**沉静射击的挂机定位**：
- 被动，按普攻次数累计自动触发，不占 AI 优先级
- 与法师的"奥秘精通"同为纯被动增伤，但游侠这版有"周期高峰"的画面感（每 N 发出现一发高伤单段）
- 与锐利强联动：锐利堆暴击率 + 暴击伤害基数 → 沉静射击这一发系数本身又高，碰上暴击时收益叠乘

**战斗公式交互**：
- 连射每段 ~65% → 有效攻击低 → 碰高甲时被双重惩罚
- 标记降低 PDEF → 改善全队物理破甲状况
- 毒箭的 DoT 可设计为固定伤害（无视破甲），给游侠对抗高甲的备选手段
- 齐射每段 ~50% → AOE 但破甲能力差 → 适合清小怪不适合打精英
- 沉静射击高系数单段 → 单发伤害远高于连射每段 → B 线对抗高甲的核心

## 5. 法师 Mage

| 技能 | 英文 | 类型 | 前置 | 分支 | 方向 | 升级效果 |
|------|------|------|------|------|------|----------|
| **魔法飞弹** | Magic Missile | 主动 | 无 | 基础 | 多枚飞弹自动追踪，根据敌人数量自适应单/群分配，低 MP 稳定输出 | 飞弹数 / 单弹伤害提升 |
| **灼热射线** | Scorching Ray | 主动 | 魔法飞弹 Lv3 | A 火 | 火系单体高系数 + 灼烧 DoT，中高 MP | 系数 / DoT 提升 |
| **寒霜射线** | Ray of Frost | 主动 | 魔法飞弹 Lv3 | B 冰 | 冰系单体中系数 + 减速 debuff（敌人 SPD 降低） | 系数 / 减速量提升 |
| **魔力** | Mana Flow | 被动 | 无 | 通用 | +MP 上限 + 每回合 MP 回复 | 回复量提升 |
| **奥秘精通** | Arcane Mastery | 被动 | 魔力 Lv3 | 通用 | +法术伤害 % | % 提升 |
| **法师护甲** | Mage Armor | 长效姿态 | 无 | 通用 | 开启时消耗少量 MP，整场战斗 +flat PDEF | +PDEF flat 值提升 |

**主动 3 : 被动 2 : 姿态 1**

**A 线（火）vs B 线（冰）的取舍**：
- A 线**纯输出**：灼热射线高系数 + 灼烧 DoT，无 debuff 干扰，打什么都疼
- B 线**控场+输出**：寒霜射线系数略低但带减速。敌人 SPD 降低 = 实际行动频率下降 = 等价于全队减伤。挂机生存性更强
- 玩家加点抉择：要纯爆发还是要稳定挂机？
- 元素亲和（怪物侧）：部分怪物有元素弱点（火怪怕冰、冰怪怕火），加点时也参考挂机地图

**魔法飞弹的自适应机制**：
- 战场单敌时：所有飞弹打同一目标 = 多段高总伤
- 战场多敌时：飞弹自动分配到不同目标 = 软 AOE
- 解决"取消爆裂后法师没 AOE"的问题，同时保持 1 环咒语的简单感
- 飞弹数随技能等级提升（DND 原版同样的成长曲线）

**法师护甲的数值守门**：
- 提供 flat PDEF 加成，每级 +X 点（X 待联调）
- 必须保证：满级法师护甲 + 轻甲 PDEF **远低于** 重甲战士 + 坚守的 PDEF
- 参考算式（Lv30 估算）：战士重甲 ~100 + 坚守 +30% = 130；法师轻甲 ~30 + 护甲满级 +20 = 50。50 << 130 ✓
- 开启代价：一次性少量 MP（约 20 MP），持续整场战斗。每场战斗只开一次 → 总 MP 时长占比极低，不影响输出循环
- 给玩家"姿态有成本"的感知，避免变成无脑常驻

**战斗公式交互**：
- 灼热射线 / 寒霜射线 / 魔法飞弹 都走 MATK × (1-MRES%)，无视 PDEF → 高甲怪的天敌
- 寒霜射线减速 → 敌方 SPD 降低 → 在 ATB 系统下等价于敌方行动间隔变长 → 实际承伤减少
- 魔力被动决定能否稳态挂机，奥秘精通决定输出上限
- 法师护甲让法师能在前线扛几下普攻，不再是"被任何小怪碰到就秒"

## 6. 圣女 Priestess

> "不要怕，有我在。"

| 技能 | 英文 | 类型 | 前置 | 分支 | 方向 | 升级效果 |
|------|------|------|------|------|------|----------|
| **神圣打击** | Smite | 主动 | 无 | 基础 | 单体魔法攻击，系数低于法师，中 MP 消耗 | 系数提升 |
| **治疗** | Heal | 主动 | 神圣打击 Lv3 | B | 回复单个队友 HP（基于 MATK） | 治疗量提升 |
| **守护祈祷** | Guardian Prayer | 被动 | 无 | B | 全队 +HP%。波次间额外回复少量 HP | % 提升 |
| **圣愈打击** | Smite of Mending | 被动 | 神圣打击 Lv3 | A | 神圣打击命中时，回复自身少量 HP 和 MP（按伤害的 X%） | 回复比例提升 |
| **神圣愤怒** | Divine Wrath | 被动 | 圣愈打击 Lv3 | A | 神圣打击 +暴击率 +暴击伤害 | % 提升 |
| **神恩** | Divine Favor | 主动 | 治疗 Lv3 | B | 给单个队友加 PATK% 或 MATK% buff（按目标职业自适应），持续数回合，中 MP / 中 CD | +攻击% / 持续时间提升 |

**主动 3 : 被动 3**

**A 线（自立）vs B 线（团辅）的取舍**：
- A 线**单刷强化**：圣愈打击让 Smite 自带回血回蓝 → 不需要切治疗 → 输出循环不中断；神圣愤怒让 Smite 暴击爆发 → 单刷效率接近法师。**单人挂机最优解**。
- B 线**组队核心**：治疗主动奶 + 守护祈祷被动团奶 + 神恩单点增益 = 完整团辅工具箱。**组队挂机必需**。
- 加点抉择：单刷玩家全点 A，组队玩家偏 B，混搭走中间路线。

**圣愈打击的设计意图**：
- 经典 WoW Smite Priest 路线——圣职业靠攻击技能续命
- 被动而非主动："吸血一击"主动版在挂机里就是 CD 好就放、日志刷屏，没意思
- 与守护祈祷形成两层续航：Smite 战中回血 + 波次间祈祷回血

**神恩的自适应机制**：
- 施给战士给 PATK%，施给法师给 MATK%
- 挂机 AI：自动挂给队伍主输出（按 法师 → 游侠 → 骑士 优先级）

**战斗公式交互**：
- 神圣打击走 MATK，无视 PDEF，但系数 ~100-110%（低于法师 150%）
- A 线圣女单人挂机时不需要切治疗自己 → 有效挂机效率接近法师
- 神恩对队友 +攻击% 是乘算 buff，对高 PATK 队友收益最大 → 强化团队核心输出

## 7. 技能前置总览

```
骑士:   重击(入口) → [Lv3] → 战吼
                   → [Lv3] → 反击 → [Lv3] → 狂怒(姿态)
        坚守(入口，独立) → [Lv3] → 守护(姿态)

游侠:   连射(入口) → [Lv3] → 标记
                   → [Lv3] → 毒箭
                   → [Lv5] → 齐射
        锐利(入口，独立) → [Lv3] → 沉静射击

法师:   魔法飞弹(入口) → [Lv3] → 灼热射线
                       → [Lv3] → 寒霜射线
        魔力(入口，独立) → [Lv3] → 奥秘精通
        法师护甲(入口，独立)

圣女:   神圣打击(入口) → [Lv3] → 治疗 → [Lv3] → 神恩
                       → [Lv3] → 圣愈打击 → [Lv3] → 神圣愤怒
        守护祈祷(入口，独立)
```

每个职业有 2-3 条线：A 线主输出延伸、B 线辅助/防御延伸、独立被动入口。前期只需要点入口技能，分支选择在 Lv5+ 后逐步显现。

## 开放问题

- [ ] TP 每级 3 点的分配方式——升级直接获取多少，外部获取多少
- [ ] 技能等级上限——10 级是否合适，和 TP 总量匹配验证
- [ ] 各技能具体系数——需要配合 combat-formula 联调
- [ ] 毒箭 DoT 是否无视破甲（固定伤害）
- [ ] 沉静射击的"每 N 次普攻"中 N 的初值和成长（影响周期爆发频率）
- [ ] 魔法飞弹的飞弹数初值与等级成长曲线
- [ ] 法师护甲的 flat PDEF 数值——必须严格联调，确保不超越战士
- [ ] 法师护甲的开启 MP 成本与持续时间（一场战斗 vs 固定时长）
- [ ] 法师 MP 消耗和魔力回复的均衡点——决定法师必须投多少点魔力才能稳态挂机
- [ ] 骑士姿态互斥的 UI 表达——挂机时如何让玩家清楚"现在开的是哪个姿态"
- [ ] 神恩"自适应队友职业"的判定逻辑——按主属性？按职业枚举？
- [ ] 圣愈打击回复量级——既要让 A 线单刷成立，又不能强到取代治疗
- [ ] 默认攻击的具体系数——法师 / 圣女魔法型默认攻击要多低，既让蓝空时有感、又不至于完全摸鱼
