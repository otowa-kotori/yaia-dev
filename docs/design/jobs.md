## 2. 技能系统规则

- **每个职业 4 个技能**（骑士暂定 3 个，第 4 个待设计）
- **每个技能可升 10 级**，每级消耗 TP
- **前置关系**：部分技能需要前置技能达到指定等级才能学习
- **TP 来源**：每级 +3 点（部分升级直接获取，部分外部获取），额外来源待定
- **TP 总量设计**：Lv30 时 TP 不足以点满所有技能。4 个技能各 10 级 = 满点 40 级，但 Lv30 总 TP 约 60-70 点（含外部来源）。需要取舍——把主力技能点到 8-10 级，还是均匀分配？不同的挂机目标（打高甲/打群怪/组队辅助）对应不同的加点策略。
- **技能重置**：低成本或免费
- **默认攻击**：所有职业都有一个不消耗资源的默认攻击（物理系普攻 / 魔法系杖击走物理）

### MP 管理（挂机模型）

```
每回合 MP 变化 = 基础回复 + 被动回复 - 技能消耗

均衡态：回复 ≥ 消耗 → 永远有技能可放（稳态挂机）
失衡态：回复 < 消耗 → MP 逐渐耗尽 → 退回默认攻击 → 效率降低
```

AI 规则：优先使用技能，MP 不够时自动用默认攻击等待回复。玩家离线前配好装备/技能/食物，确保均衡态。

- 骑士 / 游侠：MP 消耗低，基本不需要管理
- 法师：MP 消耗高，需要投资 MP 回复被动
- 圣女：MP 消耗中等，组队治疗时压力增大

## 3. 骑士 Knight

> "痛苦是骑士的勋章！"

| 技能 | 英文 | 类型 | 前置 | 方向 | 升级效果 |
|------|------|------|------|------|----------|
| **重击** | Power Attack | 主动 | 无 | 高系数单体物理，无 CD | 系数提升 |
| **战吼** | Warcry | 主动 | 重击 Lv3 | 嘲讽全体 + 自身减伤 buff，有 CD | 持续时间 / 减伤% |
| **坚守** | Fortitude | 被动 | 无 | +HP% 和 +PDEF% | % 提升 |
| **反击** | Retaliation | 被动 | 重击 Lv3 | 被攻击时概率自动反击一次普攻 | 触发概率提升 |

**主动 2 : 被动 2**

**战斗公式交互**：重击系数 >1.0 → 有效攻击高于面板 → 物理系中最容易破甲的技能。

## 4. 游侠 Ranger

> "冷静观察，一击命中。"

| 技能 | 英文 | 类型 | 前置 | 方向 | 升级效果 |
|------|------|------|------|------|----------|
| **连射** | Rapid Shot | 主动 | 无 | 多段物理（2×~65%），短 CD | 每段系数提升 |
| **标记** | Hunter's Mark | 主动 | 连射 Lv3 | 降低目标 PDEF，持续数回合，有 CD | 降低量 / 持续时间 |
| **锐利** | Keen Edge | 被动 | 无 | +暴击率 +暴击伤害 | % 提升 |
| **毒箭** | Venom Arrow | 主动 | 连射 Lv3 | 中系数单体 + DoT（持续伤害），有 CD | 系数 / DoT 伤害提升 |

**主动 3 : 被动 1**

**战斗公式交互**：
- 连射每段 ~65% → 有效攻击低 → 碰高甲时被双重惩罚
- 标记降低 PDEF → 改善全队物理破甲状况
- 毒箭的 DoT 可设计为固定伤害（无视破甲），给游侠对抗高甲的备选手段

## 5. 法师 Mage

> "只需要一发。"

| 技能 | 英文 | 类型 | 前置 | 方向 | 升级效果 |
|------|------|------|------|------|----------|
| **火球术** | Fireball | 主动 | 无 | 高系数单体魔法，高 MP 消耗 | 系数提升 |
| **爆裂** | Explosion | 主动 | 火球术 Lv3 | AOE 魔法，更高 MP 消耗，有 CD | 系数 / CD 降低 |
| **魔力** | Mana Flow | 被动 | 无 | +MP 上限 + 每回合 MP 回复 | 回复量提升 |
| **咏唱** | Incantation | 被动 | 火球术 Lv5 | 每 N 次行动触发蓄力（两段 action），下次技能伤害 ×倍率 | 倍率提升 |

**主动 2 : 被动 2**

**咏唱 AI**：触发条件满足 → 本次行动使用"蓄力"（消耗 1000 能量，给自己挂蓄力 buff）→ 下次到阈值时 intent 检测到 buff → 释放强化版技能。总成本 = 2000 能量 = 两次行动间隔。不需要额外状态机，走 intent + ability 系统。

**战斗公式交互**：
- 火球/爆裂走 MATK × (1-MRES%)，完全无视 PDEF → 高甲怪的天敌
- 魔力被动决定能否稳态挂机。不投 = 间歇性空蓝 = DPS 骤降

## 6. 圣女 Priestess

> "不要怕，有我在。"

| 技能 | 英文 | 类型 | 前置 | 方向 | 升级效果 |
|------|------|------|------|------|----------|
| **神圣打击** | Smite | 主动 | 无 | 单体魔法攻击，系数低于法师，中 MP 消耗 | 系数提升 |
| **治疗** | Heal | 主动 | 神圣打击 Lv3 | 回复单个队友 HP（基于 MATK） | 治疗量提升 |
| **守护祈祷** | Guardian Prayer | 被动 | 无 | 全队 +HP%。波次间额外回复少量 HP | % 提升 |
| **净化** | Purify | 主动 | 治疗 Lv3 | 去除队友 debuff + 小额治疗 | 治疗量 / 可净化层数 |

**主动 3 : 被动 1**

**守护祈祷的世界观**：圣女持续向神明祈祷，神力加持队伍肉体韧性（+HP%）。波次间的 HP 回复是"战斗间歇的祈祷时间"，不是战斗中凭空回血。

**战斗公式交互**：
- 神圣打击走 MATK，无视 PDEF，但系数 ~100-110%（低于法师 150%）
- 单人不需要恢复 HP 阶段（自己奶）→ 有效挂机效率接近法师

## 7. 技能前置总览

```
骑士:   重击(入口) → [Lv3] → 战吼
                   → [Lv3] → 反击
        坚守(入口，独立)

游侠:   连射(入口) → [Lv3] → 标记
                   → [Lv3] → 毒箭
        锐利(入口，独立)

法师:   火球术(入口) → [Lv3] → 爆裂
                     → [Lv5] → 咏唱
        魔力(入口，独立)

圣女:   神圣打击(入口) → [Lv3] → 治疗 → [Lv3] → 净化
        守护祈祷(入口，独立)
```

每个职业有 2 条线：一条主线（从入口技能延伸），一条独立被动。前期只需要点入口技能，不用思考太多。
