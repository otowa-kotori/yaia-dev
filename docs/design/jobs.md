## 1. 四职业总览

| 职业 | 人设 | 主属性 | 护甲 | 核心定位 |
|------|------|--------|------|----------|
| 骑士 | 金发女骑士，达克妮斯式的"挨打快乐人" | STR | 重甲 | 肉盾，承伤，嘲讽 |
| 游侠 | 精灵女弓箭手，沉稳冷静 | DEX | 中甲 | 多段输出，弱化敌人 |
| 法师 | 脆皮爆发法师，惠惠式的"一发入魂" | INT | 轻甲 | 高倍率魔法输出 |
| 圣女 | 表面娇弱但有神力加持 | INT | 中甲 | 治疗 + 神圣伤害 |

### 物理 vs 魔法

- **物理**（骑士/游侠）：受破甲线制约。面板 PATK × 技能系数 vs 敌方 PDEF → ratio-power 衰减。高系数技能更容易破甲。
- **魔法**（法师/圣女）：走 MRES% 减伤。天然能打出伤害，但 MRES 高的怪效果差。
- **核心差异**：物理天花板高但有门槛（破甲）；魔法稳定但天花板低。

## 2. 技能设计理念

### 原则

1. **每个职业 3-4 个技能**——不要多，体现特色即可
2. **每个技能都可以多次升级**（Lv 1→Lv 10+）——升级提升系数/降低 CD/增加效果
3. **有前置关系**——第一个技能是"入口"，后续技能需要前置达到一定等级才解锁
4. **技能点充裕**——选择感在"先点什么"而不是"永远放弃什么"。Lv30 时能把核心技能都点到较高级别。

### TP 来源

- 每级 +N 点（N 待定，1 或 3）
- 额外来源：副本奖励、特殊成就等（待定）
- 总量充裕——到 Lv30 时不需要痛苦取舍

### 技能升级的缩放

每级技能提升的效果应该递减（或至少线性），不能加速增长。例如：

```
伤害技能 Lv1: 120% 系数 → Lv5: 140% → Lv10: 165%
而不是 Lv1: 120% → Lv5: 200% → Lv10: 350%
```

这确保前几级投入回报最高（引导优先点核心技能），后面的等级是"锦上添花"。

### 技能重置

成本低或免费。允许玩家根据当前挂机目标调整。不惩罚尝试。

## 3. 骑士（金发女骑士）

> "痛苦是骑士的勋章！"

### 定位

前排肉盾。单人战斗靠稳定物理输出+高血量硬撑。组队时核心价值是嘲讽拉仇恨+帮后排挡伤害。

### 技能

```
[重击] ← 入口技能
  ├─→ [战吼] ← 需要重击 Lv 3
  └─→ [坚守] ← 需要重击 Lv 5
```

| 技能 | 类型 | 方向 | 升级效果 |
|------|------|------|----------|
| **重击** | 主动 | 单体高系数物理攻击（~130%→?%）| 升级提升系数 |
| **战吼** | 主动 | 嘲讽全体敌人 + 自身获得减伤 buff | 升级延长持续时间 / 提升减伤% |
| **坚守** | 被动 | 提升 HP 上限和 PDEF | 升级提升 % 加成 |

**与战斗公式的交互**：
- 重击系数高（>1.0）→ 有效攻击高 → 更容易破甲。骑士是物理系中**最能破甲**的职业
- 战吼不造成伤害但提供团队生存能力
- 坚守被动提升 PDEF → 骑士自己也更扛打

**思考**：
- 骑士的技能数量很少（3 个），但每个都有 10 级可以点。这让"点什么"变成一个有意义的资源分配问题：前期全点重击最大化挂机效率？还是早点投资战吼为组队做准备？
- 是否需要第 4 个技能？比如"盾牌反击"（被攻击时有概率反击）→ 和达克妮斯的"挨打快乐"人设匹配。但不要急着加，3 个够用先。

## 4. 游侠（精灵弓箭手）

> "冷静观察，一击命中。"

### 定位

多段物理输出。打低防目标时 DPS 最高，碰高甲目标每段都被削 → 需要配合弱化手段。

### 技能

```
[连射] ← 入口技能
  ├─→ [标记] ← 需要连射 Lv 3
  └─→ [锐利] ← 需要连射 Lv 5
```

| 技能 | 类型 | 方向 | 升级效果 |
|------|------|------|----------|
| **连射** | 主动 | 多段物理攻击（2 段 × ~65%→?%），段数固定 | 升级提升每段系数 |
| **标记** | 主动 | 标记一个敌人，降低其 PDEF，持续数回合 | 升级提升 PDEF 降低量 / 延长持续 |
| **锐利** | 被动 | 提升暴击率和暴击伤害 | 升级提升 % |

**与战斗公式的交互**：
- 连射每段 ~65% 系数 → 有效攻击 = 面板 × 0.65 → 碰高甲时破甲系数大降。**这是游侠打高甲弱的核心来源**
- 标记降低 PDEF → 直接改善**全队**的破甲状况。这是游侠的核心团队能力
- 锐利被动让暴击更频繁——暴击乘的是最终伤害，在破甲之后。低防怪时暴击收益最大

**思考**：
- 连射的段数是否应该随技能等级增加？比如 Lv1 = 2 段，Lv5 = 3 段，Lv10 = 4 段？段数增加 = 总系数上升但每段系数更低 → 高甲惩罚更重。这加深了"打软目标更强、碰高甲更弱"的分化。很有趣但要小心平衡。暂不定论。
- 标记是否需要命中才能施加？如果高甲怪也有高闪避，标记可能打不上 → 死循环。倾向标记必中。

## 5. 法师（爆发法师）

> "只需要一发。"

### 定位

魔法爆发输出。不受物防影响，天然克制高甲怪。但 HP 低、PDEF 低（轻甲），非常脆。高 MP 消耗，需要管理资源。

### 技能

```
[火球术] ← 入口技能
  ├─→ [爆裂] ← 需要火球 Lv 3
  └─→ [魔力] ← 需要火球 Lv 5
```

| 技能 | 类型 | 方向 | 升级效果 |
|------|------|------|----------|
| **火球术** | 主动 | 单体高系数魔法攻击（~150%→?%），高 MP 消耗 | 升级提升系数 |
| **爆裂** | 主动 | AOE 魔法攻击（~80%→?%），更高 MP 消耗，有 CD | 升级提升系数 / 降低 CD |
| **魔力** | 被动 | 提升 MP 上限 + 战斗中每回合回复少量 MP | 升级提升 % |

**与战斗公式的交互**：
- 火球术走 MATK × 1.5 × (1-MRES%)。不受 PDEF 影响 → 高甲怪的天敌
- 爆裂是 AOE → 打群怪效率高，但 MP 消耗大
- 魔力被动解决 MP 续航问题——没有这个被动，法师几轮就空了

**思考**：
- 惠惠式设计的核心：**单发爆炸伤害但有严重限制**。火球术的 MP 消耗应该很高，没有魔力被动时只能放几次就空。这创造了"投资魔力 vs 投资火球"的选择。
- MP 耗尽后怎么办？需要一个"默认攻击"（杖击？），走物理伤害但系数低。法师用杖击打高甲怪 = 痛苦。这逼迫法师管理 MP。
- 法师在挂机时的 MP 管理——如果 MP 耗尽要等回复，会降低挂机效率。魔力被动的投入点数直接影响挂机效率，这是一个有意义的投资决策。

## 6. 圣女（牧师）

> "不要怕，有我在。"

### 定位

治疗 + 神圣伤害双线。单人靠 Smite 练级（走 MATK 魔法伤害），伤害低于法师但有自我治疗 → 续航极强。组队时是不可替代的治疗位。

### 技能

```
[神圣打击] ← 入口技能
  ├─→ [治疗] ← 需要神圣打击 Lv 3
  └─→ [祝福] ← 需要治疗 Lv 3
```

| 技能 | 类型 | 方向 | 升级效果 |
|------|------|------|----------|
| **神圣打击** | 主动 | 单体魔法攻击（~100%→?%），系数低于法师 | 升级提升系数 |
| **治疗** | 主动 | 回复单个队友 HP（基于 MATK） | 升级提升治疗量 |
| **祝福** | 被动/光环 | 全队 HP 上限提升 + 少量每回合 HP 回复 | 升级提升 % |

**与战斗公式的交互**：
- 神圣打击 ~100% 系数 × MATK × (1-MRES%)。和法师一样无视物防，但系数低 → 单人 DPS 约法师的 60-70%
- 但圣女不需要恢复 HP 阶段（自己奶）→ **有效挂机效率接近法师**
- 治疗量基于 MATK → 投资 INT 同时提升攻击和治疗
- 祝福是团队光环 → Boss 战中大幅提升全队生存

**思考**：
- 圣女的核心矛盾：点神圣打击 = 挂机更快，点治疗 = 组队更强。前期（单人多）优先打击，后期（组队多）切治疗。技能重置免费让这个切换无痛。
- 是否需要第 4 个技能？"驱散"（去除队友 debuff）或"神圣护盾"（吸收伤害）？先不加，3 个是好的起步。
- "表面娇弱但有神力"——祝福的叙事包装：看起来是温柔的光环，实际上是全队生存的支柱。

## 7. 共享技能

暂时不做独立的共享技能池。

理由：每个职业只有 3 个技能节点，再加共享池会分散 TP 投入。在新手阶段保持简单——每个职业专注自己的 3 个技能。

如果后续需要通用能力（护甲精通、采集效率等），可以：
- 做成装备词条或全局升级，不占 TP
- 或者在一转之后引入共享池

---

## 8. 默认攻击

所有职业都有一个不消耗资源的默认攻击：

- 物理职业（骑士/游侠）：普通攻击，100% PATK，无特殊效果
- 魔法职业（法师/圣女）：杖击，100% PATK（走物理，不走 MATK）

杖击用物理管线意味着法师/圣女的默认攻击很弱（轻甲/中甲 → 低 PATK，且面对高甲怪被减伤）。这鼓励魔法职业管理 MP 而不是无脑平 A。
