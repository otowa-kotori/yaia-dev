# 新手心流总览

> 依赖：无（本文是其他模块的锚点）  
> 被依赖：progression（阶段时间线）、monsters（区域划分）

## 总目标

- 目标等级：25–30
- 日历天：约 5 天
- 前 2 天：每天 ~2 h 操作 + 挂机
- 后 3 天：每天 ~30 min 操作 + 挂机

## 阶段划分

### 阶段 0：初见（0–20 min，约 Lv 1→3）

**核心体验**：几十秒就升级，角色在变强。

- 只有战士，身上只有训练武器
- 进入第一个地图，打最弱的怪
- 2-3 波战斗后升级，弹出升级提示
- 引导去采集区域

**教学**：战斗基础、升级。

### 阶段 1：成长（20 min–2 h，约 Lv 3→8）

**核心体验**：第一次感受"制造装备→变强→打更难的怪"的核心循环。

- 学会采集 → 积累材料 → 合成第一套正式装备
- 装备后战力显著跃升
- 解锁技能点分配
- 回到战斗区打更强的怪

**教学**：采集、合成、装备、技能点。

### 阶段 2：扎根（2 h–Day 1 末，约 Lv 8→14）

**核心体验**：多角色并行的"牧场经营感"。第一次真正挂机。

- 解锁第二角色
- 装备强化系统开放
- 新地图区域开放
- 两条挂机线同时跑

**教学**：挂机循环、装备强化、多角色管理。

**追赶机制**：新角色带上装备 + 编入高等级角色队伍一起战斗 = 大量经验。不需要独立的追赶经验加成系统——组队本身就是追赶手段。

### 阶段 3：拓展（Day 1 末–Day 2 末，约 Lv 14→20）

**核心体验**：三角色稳定成长，生产链初体验。

- 解锁第三角色
- 新采集区域、更高级材料
- 第二套装备开始出现
- 教学副本（简单的固定波次副本）

**教学**：多角色协作采集/战斗、生产链、副本。

### 阶段 4：准备（Day 2–Day 4，约 Lv 20→25）

**核心体验**：遇到打不动的怪 → 思考对策 → 换装备/调技能/组队。

- 解锁第四角色
- 怪物开始有类型分化（高物防、高魔抗、高闪避等）
- 物理/魔法双伤害线的存在价值在此阶段体现
- 怪物开始有特色技能（2 图以后）
- 为 Boss 做准备

**教学**：克制关系、队伍组成策略。

### 阶段 5：决战（Day 4–Day 5，约 Lv 25→30）

**核心体验**：所有新手阶段学到的东西的综合检验。

- Boss 副本解锁
- Boss 有多阶段机制
- 需要合理的等级 + 装备 + 技能分配 + 队伍组成
- 通关 = 新手毕业

**教学**：Boss 机制、综合检验。

## 角色解锁节奏

| 解锁 | 大致时间 | 触发条件思路 |
|------|----------|-------------|
| 战士 | 开局 | 自动获得 |
| 第二角色（玩家选）| ~2 h | 战士达到一定等级或完成特定引导 |
| 第三角色（玩家选）| ~Day 1 末 | 某个角色达到更高等级 |
| 第四角色（玩家选）| ~Day 2 中 | 等级或进度门槛 |

具体解锁条件待 progression 模块确定后回填。
